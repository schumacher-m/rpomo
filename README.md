# rpomo [![Build Status](https://travis-ci.org/schumacher-m/rpomo.svg?branch=master)](https://travis-ci.org/schumacher-m/rpomo)

Pomodoro-esque CLI in Rust to be used in TMUX

## Setup

* Rust
```
cargo build --release
cargo install --force
```

## Start
```
rpomo --start
```

## Stop
```
rpomo --stop
```

## Status
```
rpomo --status
```
